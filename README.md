# Hide My Email Shortcuts

This repository contains a set of useful Shortcuts that use AppleScript to automate actions with iCloud Hide My Email.

## Shortcuts

### 1. Open Hide My Email

**Compatibility**: Mac, iPhone

**Description**: Opens the Hide My Email window in System Settings.

**Steps**:
1. Invoke the Shortcut.
2. The Hide My Email window in System System will open.

**Link**: https://www.icloud.com/shortcuts/308a86891bc94f7591d3e3a13326a26d

### 2. Search Hide My Email

**Compatibility**: Mac

**Description**: Opens Hide My Email and automatically focuses on the search bar.

**Steps**:
1. Invoke the Shortcut.
2. The Hide My Email window in System Preferences will open with the search bar in focus.

**Link**: https://www.icloud.com/shortcuts/cf9397ea0dbd4ca6b9de4d7b3752d785

### 3. Copy Hide My Email

**Compatibility**: Mac

**Description**: Takes a label as input, searches for a Hide My Email alias with the input label, and copies the alias to the clipboard if found.

**Steps**:
1. Invoke the Shortcut.
2. Input the label of the Hide My Email alias you want to find when prompted.
3. The corresponding Hide My Email alias will be copied to your clipboard if found.

**Link**: https://www.icloud.com/shortcuts/392a7289d26b430e8d5b3500a8afee6d

### 4. New Hide My Email

**Compatibility**: Mac

**Description**: Accepts a label as input, creates a new Hide My Email alias with the input label, and then copies the newly created Hide My Email alias to the clipboard.

**Steps**:
1. Invoke the Shortcut.
2. Input the label you want to give to the new Hide My Email alias when prompted.
3. A new Hide My Email alias will be created with the given label and copied to your clipboard.

**Link**: https://www.icloud.com/shortcuts/a467eec5d6e045428fb00a48d2d50ff8
